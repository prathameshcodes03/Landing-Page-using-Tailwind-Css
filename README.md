## ЁЯЪА Getting Started

### Prerequisites
- A modern web browser
- Basic knowledge of HTML, CSS, and JavaScript
- (Optional) Node.js for using Tailwind CLI

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/prathameshcodes03/Landing-Page-using-Tailwind-Css.git
   cd Landing-Page-using-Tailwind-Css
   ```

2. **Open the project:**
   - Simply open `src/index.html` in your web browser

3. **View the live page:**
   - The landing page will load immediately in your browser

## ЁЯОи Customization

### Modifying Colors
Edit the Tailwind color classes in `index.html` to match your brand palette:
```html
<!-- Change bg-blue-600 to any Tailwind color -->
<div class="bg-blue-600 text-white">
```

### Updating Content
- Edit text content directly in `src/index.html`
- Replace images in the `assets/` folder
- Update links and navigation as needed

### Adding New Sections
1. Add new HTML sections in `index.html`
2. Use Tailwind utility classes for styling
3. Add JavaScript functionality in `script.js` if needed
